"WP30 By Who" is a simple add-on that enable a small bar for label "Designed by" and social media icons links at the bottom.

== Description ==

This is first plugin from Challenge "WP30" by Terry Tsang. It aims to give credits to the maker and also the portfolio and social media link as a marketing add-on for all the WordPress sites.


There are 2 option tabs:

*   "Settings" Genera settings for E-commerce
*   "Social Options" list of social media account 


 
== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp30-by-who.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. There are new side menu "WP30" and select the submenu "By Who"
4. Update the Settings and your social media url at "Social Options" there
